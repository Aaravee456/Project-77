How to host your Visual Studio Code?



Well, to host your Visual Studio Code, first you need to download it! After dowloading it, you can customize your Visual Studio theme colors. After doing that, to make a new project/web, you need to click on "open folder", before you open your folder, make sure to have made a folder in your files app, for opening to code. After opening your folder, you can make whatever you want!



What is the console screen?



The console screen is a very important part of making your web. The console screen is like the rough work, or it is used to see if your code is working properly or not. The console screen can show you errors, mistakes and what not. You can also print some of the code you want in the console screen by using the "console.log("text")" function in your code!
